## Installation
1. Clone this repo.
2. `yarn or npm`

## How to run (dev mode)

1. `yarn dev or npm dev`

Runs the app in the development mode.

Open [http://127.0.0.1:5173/](http://127.0.0.1:5173/) to view it in the browser or seee it
on live (https://cats-and-dogs-gallery.vercel.app/)
